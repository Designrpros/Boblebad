# Boblebad Official Portal 💿

A premium, interactive single-viewport music hub and social index for Oslo-based sound designer, painter, and multi-instrumentalist **Boblebad**. Built with Next.js, React, and Tailwind CSS.

## ✨ Immersive Architectural Features

- **100vh Vertical-Coupling Turntable:** A full-height, responsive vinyl LP plate anchored precisely to the viewport boundaries featuring continuous custom rotation timelines.
- **Iridescent Lounge Backdrop:** A slow-shifting, sun-soaked Balearic morning liquid gradient mapped dynamically across the canvas coordinates.
- **Micro-House Floating Immersion:** A dense layer of 30 distinct, customized iridescent soap-bubble tracks rising from the absolute viewport baseline with synchronized delay structures.
- **Arched Brand Assets:** Replaces standardized text elements with the authentic arched logotype layout over a completely transparent background channel.
- **Minimalist Media Blocks:** Ultra-clean, border-anchored floating link matrices featuring handcrafted SVGs for key publishing nodes (Spotify, Bandcamp, Instagram, SoundCloud, YouTube, TikTok, and Facebook).

## 🛠️ Technology Stack

- **Framework:** Next.js (App Router)
- **Library:** React
- **Styles:** Tailwind CSS Engine
- **Icons:** Lucide React Icons & Handcrafted Custom Production SVGs
- **Fonts:** Geist & Geist Mono Typeface Matrices
